# 🥗 Lishe AI — Kenya Nutrition & Food Security
Healthy eating guidance, food security resources, and nutrition for all ages — in Swahili.
[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://lisheAI.streamlit.app)
## Features
- 🍽️ Personalized meal plans by age and budget
- 👶 Child nutrition and MUAC guidance
- 🌽 Local Kenyan foods and their nutritional value
- 🌾 Food security resources (WFP, UNICEF Kenya)
